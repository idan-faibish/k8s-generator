# K8s generator

A CLI tool to generate k8s resources.

## Prerequisites

1. [Node.js](https://nodejs.org) and [npm](https://npmjs.com) installed (verified with npm version 8.13.2 and Node 17.4.0).

## Installation

1. Clone this repository.
2. `npm ci`
